## 🏠 关于本站

本网站始建于2025年，我想通过搭建网站的方式，建立自己的知识体系和生活日志。

在此之前，我曾尝试过 Hexo 和 WordPress 建立博客网站 ，目前使用 [Gridea Pro](https://Gridea.pro) 同时也在 [冷汐发布](https://wechat.shang.ac.cn) 公众号进行了更新，欢迎来访，[冷汐](https://blog.shang.ac.cn) 。

我的一些课程笔记和原创的一些文章会先在订阅号发表，其次会同步到个人博客以及 --Flowus（已弃用）-- ，这个页面只用来发布关于我收藏的一些生活记录，我希望把它作为一个美好的回忆。

作为学生的我来说，我希望，它可以作为的我生活记录，永久的运营下去。前一段时间我申请了本博客的域名，它可以申请到一些优惠，我想一直维护将他作为一个博客。

这个小站承载了关于我遇见的一些美好的故事，内容可能来自网络或者任何合法渠道，如果涉及侵权之处，还望各位大佬指出，我会第一时间进行处理。侵权链接和证明材料请发送至我的邮箱。
